# JavaSimpleLoginSystem

A simple Java console application that simulates a basic login system.  
The program checks a username and password, validates input, and prints whether the login was successful.

## Features
- Username and password verification  
- Basic input validation  
- Console-based interface  
- Easy to modify and extend  

## How to Run
1. Open the project in IntelliJ IDEA (or any Java IDE)
2. Run `Main.java`
3. Enter username and password in the console

## Technologies
- Java
- IntelliJ IDEA
